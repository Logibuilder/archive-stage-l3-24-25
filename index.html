Documentation du stage - Licence 3 Math-Info

Ce document décrit les travaux réalisés durant le stage de Licence 3 MIASHS (Mathématiques et Informatique Appliquées aux Sciences Humaines et Sociales), effectué dans le cadre de l’expérimentation de techniques d’alignement d’identités dans les graphes de connaissances.

Exploration initiale et exécution des algorithmes

Dans un premier temps, j’ai exploré les dépôts GitHub officiels des différentes méthodes d’alignement d’entités (PARIS, OntoEA, CEA). Ces dépôts contiennent des jeux de données de démonstration, utilisés pour illustrer le fonctionnement de l’algorithme, ses performances et la structure attendue des fichiers d’entrée et de sortie. Les auteurs y décrivent la signification des données, les paramètres de configuration, ainsi que les résultats expérimentaux.

Mon premier objectif a donc été de réussir à faire fonctionner chaque algorithme en reproduisant leurs résultats sur ces jeux de données d'exemple. Cette étape m’a permis de me familiariser avec l’environnement d’exécution, les formats attendus, et les étapes du pipeline de traitement.

Dans un second temps, j’ai travaillé avec les deux graphes de connaissances fournis par mon tuteur. Ces données réelles, différentes des jeux de test initiaux, devaient être transformées afin d’être compatibles avec les formats d’entrée spécifiques à chaque méthode d’alignement. L’objectif était ensuite d’évaluer les performances des algorithmes d’alignement sur ces nouveaux graphes, dans un contexte applicatif concret.

Appui théorique demandé

Dans le cadre du stage, il m’a été demandé de m’appuyer sur des ressources scientifiques pour mieux comprendre les méthodes d’alignement d’identités. J’ai ainsi été amené à consulter les documents de thèse de mon tuteur, qui détaillent les fondements théoriques des approches utilisées, les problématiques liées à la représentation des entités, ainsi que les différentes stratégies d’alignement dans les graphes de connaissances.

Cette lecture m’a permis de mieux cerner les enjeux du travail demandé, de comprendre les choix algorithmiques mis en œuvre, et de faciliter l’adaptation des méthodes à mes propres données expérimentales.
